# Analise-do-Mercado-Imobiliario-POA

Esse é um projeto feito com Power BI analisando o mercado imobiliário de Porto Alegre com base no documento de IPTU da cidade do ano de 2025.
Nele é possível vermos o valor médio do m² por bairro, o valor venal médio, o total de imóveis e a média de área construída em m². Valor venal é a estimativa de preço de um bem definida pelo poder público para fins de tributação, podendo não corresponder com o valor de mercado do bem.

Também temos o total de imóveis por uso e o valor médio do m² por bairro.

Os dados foram trazidos do site da Prefeitura, por meio do portal de dados abertos, tratados por mim por meio do Colab usando pandas e python e postados no Power BI.
